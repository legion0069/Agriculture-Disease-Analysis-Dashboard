# Agriculture Disease Analysis Dashboard

## Overview
The **Agriculture Disease Analysis Dashboard** is an interactive web application built using **R Shiny** that provides insightful visualizations of crop diseases. The dashboard helps in analyzing the impact of various plant diseases across different regions using dynamic charts and maps.

## Features
- **Data Visualization**: Interactive bar charts, pie charts, scatter plots, and histograms using **Plotly** and **Highcharter**.
- **Map Integration**: Displays affected regions using **Leaflet** maps with latitude and longitude data.
- **Dynamic Data Table**: Sort and filter crop disease data efficiently with **DT**.
- **User-Friendly UI**: Intuitive and interactive interface for easy exploration of agricultural disease trends.
- **Real-World Data**: Uses real crop disease data to showcase trends and affected areas.

## Technologies Used
- **R Shiny** (Dashboard Development)
- **Plotly & Highcharter** (Data Visualization)
- **Leaflet** (Mapping)
- **DT** (Data Table)
- **Tidyverse** (Data Processing)

## Installation & Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/agriculture-disease-dashboard.git
   cd agriculture-disease-dashboard
   ```
2. Install required R packages:
   ```r
   install.packages(c("shiny", "plotly", "highcharter", "leaflet", "DT", "tidyverse"))
   ```
3. Run the Shiny app:
   ```r
   shiny::runApp()
   ```
